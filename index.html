steam-wrapper

steam-wrapper is a bash script that essentially was written to help solve
the problem on Steam on Linux distributions that were not Ubuntu Precise.
On most other distributions (like Arch Linux with which this was written
in mind), Steam may initially install successfully, but then hang on launch
without any error message or indication.

steam-wrapper runs some common "fixes" which have been noted to be successful
for various users. It does not guarantee frame rate improvements or even the
ability to launch Steam. It does however, attempt to make diagnosing problems
and launching Steam successfully easier to do.

Among other useful things, steam-wrapper provides an easy way to launch Steam
using system native libraries instead of the packaged runtime. It does not
ship with these native libraries however, and can certainly run with the
Steam runtime.

It tries to make life a bit easier for those of us who can no longer enjoy
Ubuntu Precise in 2016.
